## Installation

To set up this project locally, follow these steps:

1. Clone the repository:

   ```sh
   git clone https://github.com/aliseyedabady/node-js-blogs.git
   cd node-js-blogs
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

## Usage

To run the project, use the following command:

```sh
npm start
```
